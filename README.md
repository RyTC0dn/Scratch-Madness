# Scratch-Madness
This is a redo of a previous school project I made for level design, but this time focusing more on the gameplay aspect. Scratch Madness is a 2D bullet-hell game where you must survive an onslaught of doodle monsters. 

# Scope Statement
A 2D pixel art, bullet-hell game set in a doodle-like world where the player must bring their character to life (by drawing them) to fend off the horde of doodle monsters. Should be a short, focused experience with 1-3 levels, 3-5 weapons, and 1-3 abilities. 

# Core Gameplay Loop: 

Draw Phase:
Draw Character
Draw weapon 
Draw ability icon (as you acquire them through gameplay)
Combat Phase: 
Defeat as many enemies as possible before the countdown runs out
Collect as many points as possible by defeating enemies
Progress to the next level
In-between level phase:
Purchase new weapons or abilities 
Upgrade equipment (weapons, abilities, armor)

*Adjust scope while in production
