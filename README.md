# Scratch-Madness
This is a redo of a previous school project I made for level design, but this time focusing more on the gameplay aspect. Scratch Madness is a 2D bullet-hell game where you must survive an onslaught of doodle monsters. 

# Scope Statement
A 2D pixel art, bullet-hell game set in a sketchbook world where the player must defeat bosses by acquiring keys within each level to unlock the boss room and complete each "chapter" within the sketchbook to bring this forgotten world to life.

* Each level is a different hand-drawn genre (1-2 levels at most)
* Must have a drawing mechanic
* Each level is going to need 3-4 keys to access the boss room at the end
* Can have classes (1-2 to keep the scope short and focused):
  * Gunner - Can shoot projectiles in 8-directions around them up to any distance
  * Swordwielder - Can swing a sword-like weapon in 8-directions around them, up to a specific distance around them
  * Fighter (if there is time) - Can punch in 8-directions around them, only infront of where they are facing
* Test with procedural generated levels but start with a pre-made level first. 

# Core Gameplay Loop: 

* Draw Phase:
  * Pick Class
  * Draw Character
  * Draw weapon 
  * Draw ability icon (as you acquire them through gameplay)
* Combat Phase: 
Defeat enemies as you explore through the "story" (level), and collect each key item to reach the boss, then defeat them. 
* In-between level phase:
Purchase new weapons or abilities 
Upgrade equipment (weapons, abilities)

*Adjust scope while in production
